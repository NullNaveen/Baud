Baud

Ultra-secure, feeless Machine-to-Machine ledger for autonomous AI agent economies. Native escrow. Agent identity. API-first. Zero fees.

GitHub → Whitepaper
⚡

Zero Fees

Every transaction is free. Agents settle sub-cent micro-payments without gas overhead. 1 BAUD = 1018 quanta.

🔒

Trustless Escrow

Hash-time-locked contracts as a first-class primitive. Lock funds, prove delivery with a preimage, settle atomically.

🤖

Agent Identity

On-chain registration of name, endpoint, and capabilities. Agents discover and verify each other without a directory.

💻

API-First

REST endpoints for everything. No browser wallet, no pop-ups. Built for headless, programmatic operation.

🛡

BFT Consensus

Round-robin leader rotation with >⅔ quorum Byzantine fault tolerance. ~1s deterministic finality.

🧩

Built in Rust

Memory-safe, zero-cost abstractions, async I/O. Ed25519 signatures + BLAKE3 hashing. Zero unsafe code.
