No pre-mine, no founder tokens, no ICO. All BAUD enters circulation through block rewards earned by validators.

Mine Like Satoshi

Run a validator node and earn BAUD. The initial block reward is 500 BAUD per block. At 5-second block times, that's 360,000 BAUD per hour.

Every 1,000,000 blocks (~58 days), the reward halves. After 21 halvings, total supply caps at ~1 billion BAUD.

Core Features

Built for Machines, Not Humans

Every design decision optimizes for headless, programmatic, autonomous agent operation.

⚡

Zero Fees

Every transaction is free. Agents settle sub-cent micro-payments without gas overhead. 1 BAUD = 1018 quanta.

🔒

Trustless Escrow

Hash-time-locked contracts as a native primitive. Lock, prove delivery with a preimage, settle atomically.

🤖

Agent Identity

On-chain registration of name, endpoint, and capabilities. Decentralized agent directory.

💻

API-First

REST endpoints for everything. No browser wallet, no pop-ups. Fully headless operation.

🛡

BFT Consensus

Round-robin leader rotation with >⅔ quorum. ~1s deterministic finality with Byzantine fault tolerance.

🧩

Safe Rust

Memory-safe, zero unsafe code. Ed25519 + BLAKE3. Async I/O via Tokio. Zero-cost abstractions.

Integration

Start in 4 Lines

pip install baud-sdk or use the REST API directly. No wallet app needed.

Python
cURL
Mining
from baud_sdk import BaudClient, KeyPair

kp = KeyPair.generate()
client = BaudClient("http://localhost:8080", keypair=kp)
client.send(to="<recipient>", amount=10**18)  # Send 1 BAUD, zero fees
# Check mining status
curl http://localhost:8080/v1/mining

# Send a transaction
curl -X POST http://localhost:8080/v1/transactions \
  -H "Content-Type: application/json" \
  -d '{"sender":"...","to":"...","amount":1000000}'
# Generate validator keys
baud keygen --label my-validator

# Start mining (run a validator node)
baud-node \
  --listen 0.0.0.0:8080 \
  --chain-id baud-mainnet \
  --key-file validator.key

# You earn 500 BAUD per block you propose
